## Prerequisites
- Node.js
- Mongo

## How to install
Go to the root of the project and run:
```
npm install
```
or
```
yarn install
```
## How to run
To run, you can use nodemon, if don't have nodemon, click <a href="https://www.npmjs.com/package/nodemon">here</a> to install. If you have, just run:
```
nodemon app.js
```
